Estrucutra básica de CSS

Código CSS:

selector{
    propiedad: valor;
}

Este es un ejemplo de código CSS:

h1{
    background-color: red;
}

Selectores básicos

  1. De tipo - div{ ... }
  2. De clase - .elemento { ... }
  3. De ID - #id-del-elemento { ... }
  4. De atributo - a[href= "..."] { ... }
  5. Universal - * { ... }
