Cascada y especifidad

Cascada:

El orden del código CSS importa, el IDE lee el CSS de arriba hacia abajo y en caso de requerirlo sobre escribe el código de una etiqueta

Por ejemplo el h2 de: Cascada tiene dos colores rojo y azul pero como azul está más abajo en el código éste prevalece sobre el rojo

Especifidad:

La especifidad en un cpodigo CSS sirve para que el navegador pueda decidir la prioridad de los estilos de un documento
