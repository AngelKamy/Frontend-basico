Estas son las etiquetas más utilizadas en HTML:

    Layout

    Estas etiquetas tienen una función principalmente semántica y son las etiquetas que le darán la estructura principal a nuestro documento

  • header - Es el encabezado de la página web
  • nav - En ésta etiqueta se pone el navegador interno de la página web
  • section - Se utiliza para asignar una sección específica donde se pondrán elementos
  • article - Se utiliza principalmente para designar entradas de blog y artículos
  • aside - Se utiliza para designar una barra lateral dentro del sitio web
  • footer - En esta sección se pone el pie de página

    Enlaces

    Estas etiquetas se utilizan para crear enlaces externos a otras páginas, sitios o elementos

  • a - Es una etiqueta de auto cierre y se utiliza para vincular elementos de la red

    Textos

    Estas etiquetas se utilizan para crear textos con una funcionalidad específica

  • h1 ... h6 - Estas etiquetas se utilizan para crear los títulos y encabezados, siendo h1 el de mayor rango y h6 al de menor rango
  • p - Es la etiqueta de texto genérico se utiliza para darle cuerpo al párrafo
  • span - Es un elemento de texto especial, y por lo tanto en línea.

    Imágenes y videos

    Son las etiquetas que asignan elementos audiovisuales

    Formularios

    Los formularios tienen etiquetas específicas para poder funcionar dentro de HTML

  • form - Esta etiqueta crea la sección principal de un formulario
  • input - Crea una entrada de datos para el usuario, se debe utilizar un atributo específico para darle formato a la entrada
  • label - Acompaña a la etiqueta de input y sirve como texto informativo
  • button - Crea un input específico en forma de botón

    Listas

    Son las etiquetas que estructuran las listas dentro de html

  • ul - Crea una estrucutra de lista sin un orden específico
  • ol -Crea una estrucutra de lista con un orden específico
  • li -Es un elemento dentro de la estructura de lista
